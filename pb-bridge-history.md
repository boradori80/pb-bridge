# 🚀 PB-Bridge 개발 학습 로드맵

### [Day 1] 2026-05-20: 시작 (Hello World)
- **주요 작업:** GitHub 저장소 생성 및 README 작성
- **학습 내용:** Git의 개념 (내 컴퓨터와 클라우드의 동기화)

### [Day 2] 2026-05-21: 뼈대 구축 (Next.js Infra)
- **주요 작업:** Next.js 설치 및 환경 설정
- **학습 내용:** 터미널 명령어(CLI) 사용법, D드라이브 권한 문제 해결(dubious ownership)

### [Day 3] 2026-05-22: 첫 화면 (UI Design)
- **주요 작업:** 다크모드 대시보드 레이아웃 구현
- **학습 내용:** Tailwind CSS (미리 정의된 클래스로 웹 디자인하는 법), Vim 에디터 탈출

### [Day 4] 2026-05-23: 파일 인식 (File Reader)
- **주요 작업:** 드래그 앤 드롭으로 `.srd` 파일 읽기 성공
- **학습 내용:** FileReader API (브라우저가 내 PC 파일을 읽어오는 보안 원리)

### [Day 5] 2026-05-24: 문장 해석 (Basic Parsing)
- **주요 작업:** 파일 헤더에서 'release' 버전 및 메타데이터 추출
- **학습 내용:** 정규표현식(Regex)을 통한 데이터 구조화, React의 데이터 바인딩 원리

### [Day 6] 2026-05-25: 컬럼 추출 및 반복문 바인딩
- **주요 작업:** `.srd` table 내의 column 속성(name, type) 파싱 및 테이블 UI 출력
- **학습 내용:** RegExp의 글로벌 매칭, JavaScript의 배열 구조화, React `.map()` 함수를 이용한 동적 UI 렌더링 원리 (파워빌더의 For문+InsertRow 메커니즘과 비교)

### [Day 7] 2026-05-26: 조회 인자(Retrieval Arguments) 추출 및 시각화
- **주요 작업:** `.srd` 파일에서 `arguments=(...)` 구문 파싱 기능 추가 및 우측 조회 인자 명세 UI 구현
- **학습 내용:** 중첩 괄호 문자열을 처리하는 자바스크립트 정규표현식의 탐색 원리, React에서 논리곱(`&&`) 연산자를 통한 조건부 렌더링 및 배열 `.map()`을 활용한 UI 동적 데이터 바인딩

### [Day 8] 2026-05-29: 데이터윈도우 SQL Retrieve 구문 파싱 완료 
- **주요 작업:** `.srd` 파일 내의 retrieve 속성(SQL문) 추출 및 전용 쿼리 뷰어 UI 구현
- **학습 내용:** 줄바꿈을 포함한 대용량 문자열을 매칭하는 정규표현식 패턴 공부, HTML `<pre>` 태그를 활용한 소스코드 포맷팅 출력 원리 이해

### [Day 8] 2026-05-29: SQL 구문 강조(Syntax Highlighting) 엔진 구현 
- **주요 작업:** 추출된 SQL 내부의 DB 예약어 정규식 치환 및 하이라이팅 UI 구현
- **학습 내용:** React의 dangerouslySetInnerHTML 원리 이해, 정규표현식을 이용한 토큰(Token) 강조 메커니즘 학습

### [Day 9] 2026-05-30: SQL 정렬(Formatting) 기능 구현
- **주요 작업:** 추출된 SQL 쿼리를 보기 좋게 줄바꿈과 들여쓰기로 자동 정렬하는 기능 추가 및 UI 개선
- **학습 내용:** 정규표현식을 이용한 토큰(Token) 강조 메커니즘 학습

### [Day 10] 2026-05-31: 데이터윈도우 컬럼 기반 동적 그리드 프리뷰 구현
- **주요 작업:** 파싱된 Column 배열 데이터를 활용한 가변형 HTML 테이블 헤더 생성 및 샘플 데이터 바인딩 프리뷰 UI 구현
- **학습 내용:** React의 동적 렌더링(Dynamic Rendering) 제어 기법 학습, 데이터 구조가 화면 컴포넌트로 자동 변환되는 웹 컴포넌트 핵심 메커니즘 이해

### [Day 11] 2026-06-01: 컬럼 디자인 메타데이터(라벨, 정렬) 파싱 및 프리뷰 연동
- **주요 작업:** `.srd` 헤더 텍스트 오브젝트 및 정렬 속성 추출 엔진 구현, 동적 프리뷰 UI에 한글 라벨 및 CSS 정렬 바인딩
- **학습 내용:** 값 변환을 위한 JavaScript 매핑(Mapping) 기술 학습, 리액트 `className`을 이용한 동적 스타일링(Dynamic Styling) 제어 메커니즘 이해


### [Day 12] 2026-06-02: 컬럼 데이터 타입별 동적 입력 폼(Form) 프리뷰 엔진 구현 ◀ 진행 중
- **주요 작업:** `.srd` 파일의 데이터 타입(char, datetime, number)을 분석하여 HTML5 맞춤형 입력 컴포넌트 동적 매핑 및 Form UI 구현
- **학습 내용:** 소스코드 내 주석에 작업 일자(Day 12) 기록 관리 적용, React의 조건부 렌더링(Conditional Rendering)을 통한 동적 웹 컴포넌트 생성 원리 학습
