# 🚀 PB-Bridge 개발 대여정 및 마이그레이션 역사 기록 (Day 1 ~ Day 15)

본 문서는 파워빌더(PowerBuilder) 레거시 시스템 소스코드를 현대적인 웹 표준 기술(Next.js, TypeScript, Tailwind CSS)로 정교하게 자동 변환하는 'PB-Bridge' 솔루션의 일별 연구 개발 및 학습 일지입니다.

---

### [Day 1] 2026-05-22: 프로젝트 뼈대 구축 및 파일 파서(Parser) 기초 설계
- **주요 작업:** Next.js 기반 다크모드 대시보드 레이아웃 UI 설계 및 기본 파일 업로드 컴포넌트 배치
- **학습 내용:** React의 컴포넌트 구조 이해, Tailwind CSS를 이용한 유연한 대시보드 테마 스타일링 학습

### [Day 2] 2026-05-23: 파워빌더 Export Header 및 버전 감지 엔진 구현
- **주요 작업:** 업로드된 파일 내 `$PBExportHeader$` 및 `release` 구문을 분석하여 파워빌더 버전을 자동으로 판별하는 파싱 로직 구현
- **학습 내용:** 자바스크립트 `String.prototype.match()` 함수와 정규표현식(`regex`)의 첫 번째 캡처 그룹 활용 원리 학습

### [Day 3] 2026-05-24: Window/UserObject UI 컨트롤 추출 엔진 구현
- **주요 작업:** `.srw`, `.sru` 파일 구조 분석을 통해 `type [컨트롤명] from [상속타입]` 패턴을 추적하여 UI 부품 목록을 자동 추출하는 엔진 빌드
- **학습 내용:** 정규식 전역 검색 플래그(`/g`)와 `RegExp.prototype.exec()` 메서드를 활용한 텍스트 다중 매칭 메커니즘 체득

### [Day 4] 2026-05-25: 환경 구축 최적화 및 원격 저장소 동기화
- **주요 작업:** 로컬 개발 인프라 안정화 작업 진행, 깃허브(GitHub) 원격 레포지토리 연결 및 형상 관리 체계 확립
- **학습 내용:** `git init`, `git remote add`, `git push` 등 분산 버전 관리 시스템의 핵심 명령어 구조 마스터

### [Day 5] 2026-05-26: DataWindow 레이아웃 밴드(Band) 높이 추출 기술 구현
- **주요 작업:** 데이터윈도우 `.srd` 소스 내부의 `header(height=...)`, `detail(height=...)` 등 각 구역별 고유 픽셀 높이를 정규식으로 정밀 추출하는 로직 구현
- **학습 내용:** 동적으로 정규식 객체를 생성하는 `new RegExp()` 활용법 및 문자열을 안전하게 정수로 바꾸는 `parseInt(text, 10)` 진수 설정 학습

### [Day 6] 2026-05-27: DataWindow 뼈대 컬럼(Column) 사양 분석 엔진 구현
- **주요 작업:** `table(column=(...))` 블록 내부를 정밀 샌드박싱하여 영어 컬럼명, 데이터 타입, DB 바인딩 명세를 완벽히 분리하는 핵심 파서 구축
- **학습 내용:** 괄호 안의 중첩 구조(`char(10)`, `decimal(4)`)가 잘리지 않도록 방어하는 비탐욕적(Non-greedy) 중첩 정규표현식 대응 패턴 기술 습득

### [Day 7] 2026-05-28: SQL Retrieve 조회 쿼리 원문 추출 파서 구현
- **주요 작업:** 데이터윈도우의 심장인 `retrieve="..."` 구문 내부의 다중행 SQL 쿼리를 완벽하게 추출하는 엔진 구현
- **학습 내용:** 개행 문자(\n)까지 세상의 모든 문자를 빠짐없이 탐색하는 `[\s\S]*?` 비탐욕적 매칭의 원리 및 파워빌더 고유의 이스케이프 따옴표(`~"`)를 웹 표준(`"`)으로 변환하는 치환 처리 학습

### [Day 8] 2026-05-29: 오라클 토드 스타일 SQL 구문 강조(Syntax Highlighting) 구현
- **주요 작업:** 추출된 SQL 내 DB 주요 예약어(SELECT, FROM, WHERE 등)를 실시간 감지하여 알록달록하게 색상을 입혀주는 구문 강조 엔진 구현
- **학습 내용:** React 자동 보안 필터를 우회하여 HTML 태그를 진짜 화면 부품으로 그려내는 `dangerouslySetInnerHTML` 속성의 구동 원리 및 XSS 해킹 취약점 방어를 위한 사전 샌드박스 안전화(Escaping) 기술 마스터

### [Day 9] 2026-05-30: 버튼 클릭식 SQL 자동 정렬 포맷터(Formatter) 구현
- **주요 작업:** 한 줄로 길게 늘어진 스파게티 SQL 쿼리를 규칙 기반으로 자동 줄바꿈 및 4칸 들여쓰기(Indentation) 해주는 포맷터 버튼 및 토글 UI 구현
- **학습 내용:** CSS 스타일 `whitespace-pre-wrap`이 지정된 `<pre>` 태그 내부에서 줄바꿈 문자(`\n`)와 공백이 왜곡 없이 브라우저에 그대로 재현되는 원리 학습

### [Day 10] 2026-05-31: 컬럼 리스트 기반 동적 데이터 그리드 프리뷰(Grid Preview) 구현
- **주요 작업:** 파싱된 Column 배열 데이터를 그대로 재활용하여 가로 헤더를 만들고 임시 샘플 데이터 3줄을 채워 즉석에서 테이블 UI를 그려내는 프리뷰 엔진 구현
- **학습 내용:** 고정된 표가 아니라 데이터 개수에 따라 가변형 UI를 선언적으로 루프 도는 React 고유의 `columns.map()` 반복문 바인딩 메커니즘 학습

### [Day 11] 2026-06-01: 파워빌더 한글 헥사 코드 디코더 및 정렬(Align) 연동 완료
- **주요 작업:** 파워빌더 소스 특유의 깨진 한글 패턴(`$$HEXn$$...$$ENDHEX$$`)을 완벽한 한글로 복원하는 디코더 함수 구현 및 정렬 속성(`alignment`)을 Tailwind CSS 클래스로 매핑 연동
- **학습 내용:** 16진수 문자열을 4글자 단위로 쪼개어 바이트 스왑(Byte Swap) 후 UTF-16LE 기반 한글 정수로 디코딩하는 원리 학습, 객체 매핑(Mapping Object) 기법을 통한 최적화 학습

### [Day 12] 2026-06-02: 데이터 타입별 맞춤형 웹 입력 폼(Form Preview) 구현
- **주요 작업:** 컬럼 사양의 `char`, `datetime`, `number` 타입을 분석하여 일반 입력창, 달력 팝업, 숫자 전용 인풋으로 동적으로 컴포넌트를 갈아끼우는 폼 프리뷰 카드 레이아웃 구현
- **학습 내용:** 자바스크립트 `switch-case` 분기를 통해 HTML 부품 자체를 변수처럼 다루어 반환하는 React 조건부 렌더링(Conditional Rendering) 원리 학습

### [Day 13] 2026-06-03: DataWindow Expression(연산 필드) 파싱 및 실시간 화면 연산 구현
- **주요 작업:** `compute(...)` 구문 속 계산식 원문(`salary * 1.1`)을 정규식으로 추출하여, 사용자가 프리뷰 값을 바꿀 때마다 실시간 연산 결과를 화면에 갱신하는 엔진 구현
- **학습 내용:** `onChange` 이벤트 핸들링을 통한 실시간 값 포착 원리, 천단위 콤마 문자열 포맷팅(`toLocaleString`) 처리와 순수 수치 계산을 상호 변환하는 상태 제어 기법 학습

### [Day 14] 2026-06-04: 파싱 엔진 영역별 예외 처리(Try-Catch) 및 에러 방어 UI 구현
- **주요 작업:** 파서 내부 주요 로직들을 개별 `try-catch` 블록으로 격리하여 특정 영역 구문 손상 시 전체 화면이 멈추는 크래시 방지 및 다크 로즈 테마 에러 피드백 UI 레이어 구현
- **학습 내용:** 자바스크립트 런타임 예외 처리(Exception Handling) 메커니즘 학습 및 손상된 데모 파일(`dw_corrupt_test.srd`)을 이용한 결함 허용(Fault Tolerance) 시스템 설계 검증 완료

### [Day 15] 2026-06-05: 아키텍처 고도화를 위한 소스코드 모듈 분할 및 토큰 최적화 완료 ◀ 진행 중
